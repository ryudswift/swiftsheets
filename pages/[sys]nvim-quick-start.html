Neovim Quick-Start Guide

Your first steps into a more powerful editor. No customization, just pure commands.

The Absolute Essentials

Understanding Modes

Neovim is a modal editor. You start in Normal Mode for navigation and commands. To type, you must enter Insert Mode.

  • i → Insert text before the cursor.
  • a → Append text after the cursor.
  • Esc → Return to Normal Mode. This is your most important key!

Saving & Quitting

All these commands start with a colon (:) in Normal Mode.

  • :w → write (save) the file.
  • :q → quit the editor.
  • :wq → Write and quit (save & exit).
  • :q! → Quit without saving. The '!' forces the action.

Basic Navigation & Editing

All commands are used from Normal Mode. Practice these until they become muscle memory.

Character Movement

  • h → Move left
  • j → Move down
  • k → Move up
  • l → Move right

Word & Line Movement

  • w → Forward one word
  • b → Backward one word
  • 0 → Start of the line
  • $ → End of the line

File Movement

  • gg → Go to top of file
  • G → Go to bottom of file
  • 5G → Go to line 5 (number + G)

Deletion

  • x → Delete character
  • dd → Delete current line

Copy & Paste

  • yy → Yank (copy) line
  • p → Paste after cursor

Undo / Redo

  • u → Undo
  • Ctrl + r → Redo

The Vim Grammar: Verb + Noun

Combine verbs (actions) with nouns (motions/text objects) for powerful, precise edits.

Verb + Noun = Action

Verbs (Operators)

  • d → delete
  • c → change (delete then insert)
  • y → yank (copy)

Nouns (Motions)

  • w → word
  • $ → to end of line
  • iw → inner word
  • i) → inside parentheses

Examples in Action:

  • dw → delete word
  • c$ → change to end of line
  • di) → delete inside )

Intermediate Techniques

Search

  • / + term + Enter → Search forward
  • ? + term + Enter → Search backward
  • n → Jump to next match
  • N → Jump to previous match

Visual Mode & Window Splits

  • v → Enter Visual mode (select text)
  • :sp → Split window horizontally
  • :vsp → Vertically split window
  • Ctrl+w h/j/k/l → Navigate splits

The Dot Operator: Your Superpower

The dot (.) command is one of Vim's most powerful features. It repeats the last change you made.

Imagine you deleted a word with dw. You can now go to any other word and press . to delete it too. This works for any change, like ciw, x, etc.

Master these commands to build a solid foundation. Happy Vimming!
